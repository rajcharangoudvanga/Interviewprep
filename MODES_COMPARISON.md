## Which Mode Should I Use?

### Use **Text Mode** if you:
- Want to carefully craft responses
- Are practicing technical/coding questions
- Need to include code snippets or diagrams
- Are in a public/noisy environment
- Don't have a microphone
- Want maximum browser compatibility

### Use **Voice Mode** if you:
- Want realistic interview practice
- Are practicing behavioral questions
- Want to improve verbal communication
- Have a quiet environment
- Have a good microphone
- Want to practice speaking clearly

### Use **Both** for:
- Complete interview preparation
- Balanced skill development
- Different question types
- Maximum practice variety

## Tips for Each Mode

### Text Mode Tips
1. **Structure your answers** with clear paragraphs
2. **Use bullet points** for lists
3. **Include examples** with specific details
4. **Proofread** before submitting
5. **Take your time** - no rush!

### Voice Mode Tips
1. **Speak clearly** at normal pace
2. **Pause between thoughts** for better transcription
3. **Review transcript** before submitting
4. **Practice in quiet space** for best results
5. **Use natural language** - speak conversationally

## Common Questions

**Q: Can I use both modes in the same interview?**
A: No, but you can practice the same role/level in both modes separately.

**Q: Which mode gives better feedback?**
A: Both modes provide identical feedback quality. The difference is only in input method.

**Q: Is voice mode as accurate as typing?**
A: Yes! Modern speech recognition is very accurate. You can edit the transcript if needed.

**Q: Does voice mode work on mobile?**
A: Yes! Works great on iOS Safari and Android Chrome.

**Q: Which mode is more realistic?**
A: Voice mode is more realistic for actual interviews, but text mode is better for technical depth.

**Q: Can I practice the same questions in both modes?**
A: Questions are generated fresh each session, but you'll get similar types of questions for the same role/level.
